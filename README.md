# generevideo
Ce script permet de générer une vidéo à partir des images se trouvant dans le même répertoire, peu importe le format.
Si en plus il trouve des fichiers .mp3, il ajoutera une bande son.
C'est très basique mais pratique pour générer une rapide présentation, sans fioritures, simplement.
generevideo.sh a été créé à moitié avec Gemini, à moitié en codant en manuel pour rectifier les inévitables gags de l'IA ;-)
